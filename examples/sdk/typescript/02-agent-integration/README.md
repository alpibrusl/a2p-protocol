# Example 02: Agent Integration

This example demonstrates how to build an AI agent that uses a2p profiles.

## What You'll Learn

- Creating an agent client
- Requesting access to user profiles
- Using profile data in prompts
- Proposing new memories

## Run TypeScript Version

```bash
pnpm install
pnpm start
```

## Run Python Version

```bash
npm install
npm run 02:*
```
